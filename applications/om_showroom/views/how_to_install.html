How to Install Games from OM-Showroom

Use the following instructions:

  1. Open a shell session in your fabourite terminal or though SSH

  2. If your are using SHR:

    1. opkg update
    2. opkg install The_Name_of_app_to_install_here

  3. If your are using Debian:

    1. apt-get udate
    2. apt-get install The_Name_of_app_to_install_here
